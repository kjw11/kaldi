
This directory (cn-celeb) contains baseline system scripts for CN-Celeb
 dataset. CN-Celeb corpus is required, which can be download in:
xxxxxxxx

For more details about this dataset, please refer paper "CN-CELEB: A 
CHALLENGING CHINESE SPEAKER RECOGNITION DATASET", paper url:
xxxxxxxx

This directory contains two baseline system, v1 is a i-vector based system 
while v2 is a x-vector based system. Note that the CN-Celeb dataset does not 
give a fixed training set and evaluation set, the recommended training 
speaker list and evalation speaker list are located in v1/local/train_id_list 
and v1/local/train_id_list, which can be replaced to make adaptation.
